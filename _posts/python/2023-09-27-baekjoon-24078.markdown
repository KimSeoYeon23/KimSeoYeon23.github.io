## 백준 알고리즘 24078번 余り (Remainder)

### 문제

양의 정수 X가 주어졌을 때, X를 21로 나눈 나머지를 출력하라.

### 입력

입력은 표준 입력에서 다음과 같은 형식으로 제공된다.

```
X
```

### 출력

X를 21로 나눈 나머지를 출력하라.

### 제한

- 100 <= X <= 100/
- X는 정수이다.

### 예제

| 입력 | 출력 |
| --- | --- |
| `50` | `8` |
| `42` | `0` |
| `5` | `5` |

### 풀이

```py
X = int(input())
print(X % 21)
```
