## 백준 알고리즘 10171번

고양이를 출력한다.

**예제 출력**

```text
\    /\
 )  ( ')
(  /  )
 \(__)|
```

**풀이**

```py
print("\\    /\\")
print(" )  ( ')")
print("(  /  )")
print(" \\(__)|")
```

역슬래시('\')는 파이썬에서 escape 문자로 사용되므로 두 번 써서 실제로 하나의 역슬래시를 출력한다.
