## 백준 알고리즘 8393번 합

### 문제

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

### 입력

첫째 출에 n (1 <= n <= 10,000)이 주어진다.

### 출력

1부터 n까지 합을 출력한다.

### 예제

| 입력 | 출력 |
| :--: | :--: |
| `3` | `6` |

### 풀이

```py
n = int(input())
result = 0

for i in range(n):
  result += i+1
print(result)
```
