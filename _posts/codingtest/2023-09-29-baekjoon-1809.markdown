### 문제

### 예제

```text
(___)
(o o)____/
 @@      \
  \ ____,/
  //   //
 ^^   ^^
```

### 풀이

```Golfscript
"(___)
(o o)____/
 @@      \\
  \\ ____,/
  //   //
 ^^   ^^"
```
