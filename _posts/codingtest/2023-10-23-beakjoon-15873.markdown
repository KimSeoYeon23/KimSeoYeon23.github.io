## 백준 알고리즘 15873번 공백 없는 A+B

### 문제

자연수 A, B가 주어지면 A+B를 구하는 프로그램을 작성하시오.

### 입력

자연수 A, B (0 < A, B <= 10)가 첫 번쨰 줄에 주어진다. 단, 두 수의 사이에는 공백이 주어지지 않는다. 두 수의 앞에 불필요한 0이 붙는 경우는 없다.

### 출력

첫 번째 줄에 A+B의 값을 출력한다.

### 예제

| 입력 | 출력 |
| --- | --- |
| `37` | `10` |
| `102` | `12` |

### 풀이

```py
s = input()

if s[1] == "0":
  print(10 + int(s[2:]))
else:
  print(int(s[0]) + int(s[1:]))
```
