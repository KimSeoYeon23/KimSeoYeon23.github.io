## 백준 알고리즘 2440번 별 찍기 - 3

### 문제

첫째 줄에는 별 N개, 둘째 줄에는 별 N-1 개, ..., N번째 줄에는 별 1개를 찍는 문제

### 입력

첫째 줄에 N(1 <= N <= 100)이 주어진다.

### 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

### 예제

| 입력 | 출력 |
| --- | --- |
| `5` | `*****` <br/> `****` <br/> `***` <br/> `**` <br/> `*` |

### 풀이

```py
n = int(input())

for i in range(n, 0, -1):
  print("*" * i)
```
