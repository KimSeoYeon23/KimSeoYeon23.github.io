## 백준 알고리즘 11654번 아스키 코드

### 문제

알파벳 소문자, 대문자, 숫자 0-9 중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

### 입력

알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

### 출력

입력으로 주어진 글자의 아스키 코드 값을 출력한다.

### 예제

| 입력 | 출력 |
| :--: | :--: |
| `A` | `65` |
| `C` | `67` |
| `0` | `48` |
| `9` | `57` |
| `a` | `97` |
| `z` | `122` |

### 풀이

```py
print(ord(input()))
```

1. `ord()` 함수는 문자를 아스키 코드로 반환해준다.
